# Frontend Folder Note

The working frontend code is currently stored in the project root:

- `src/`
- `index.html`
- `package.json`
- `vite.config.js`
- `tailwind.config.js`
- `postcss.config.js`

These files were not moved because the app currently runs from the root using:

```powershell
npm.cmd run dev -- --port 3000
```

Moving them into this folder would require updating project paths and run commands.
